E-Books-Catalog

To start app
```commandline
make run
```

To do migrations
```commandline
make migrations
make migrate
```

Base env in default.env, copy it to .env

User can CRUD (create, read, update, delete) operations with:
 - books
 - genres
 - reserve

Swagger - {{host}}/docs

Work with docker compose
```commandline
cd app
docker compose build
docker compose up
```
